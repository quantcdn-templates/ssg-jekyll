This is the base Jekyll theme. You can find out more info about customizing your Jekyll theme, as well as basic Jekyll usage documentation at [jekyllrb.com](https://jekyllrb.com/).

## About This Template

This Jekyll template is configured for deployment to QuantCDN, providing:

- **Global CDN Distribution**: Your static site served from edge locations worldwide
- **Automatic Builds**: GitHub Actions workflow for continuous deployment
- **Cache Purging**: Automatic cache invalidation on new deployments
- **SSL/TLS**: Secure HTTPS connections out of the box

## Getting Started

1. Fork or clone this repository
2. Configure your Quant secrets in GitHub
3. Push changes to trigger a deployment
